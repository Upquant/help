# 优品量化平台

> <i class="fa fa-quote-left fa-3x pull-left fa-border"></i>
优品量化平台是优品财富旗下的一款多元资产量化交易平台，参考国外投行量化平台的设计模式，完全自主研发，专注于为专业投资者提供的可靠、高效的服务。优品量化以海量金融数据为基础，提供策略研发、回测、模拟交易和实盘交易等功能，支持股票、期货等多账户多品种的交易，为投资者打造一个专属的量化平台。
